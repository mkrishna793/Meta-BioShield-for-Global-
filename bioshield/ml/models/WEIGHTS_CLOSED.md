# Model Weights — CLOSED

The trained model weights (`rf_model.joblib`, `xgb_model.joblib`, `meta_learner.joblib`) are **not open-sourced**.

## Why?

Publishing trained weights creates a "White Box" that allows adversarial AI to test millions of evasion sequences offline. By keeping weights closed, bad actors must submit real DNA orders to test the system — triggering cost, speed, and legal traps.

## Access

Model weights are deployed as a closed API within trusted screening networks. For research access, contact the team.
